# TinyApp Project

TinyApp is a full stack webb application built with Node and Express that allows users to shorten long URLs (a la bit.ly)

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
